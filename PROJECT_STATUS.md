## 🏗️ 项目架构

### 系统组成

```
elderly-care-system/
├── backend/              # Go 后端服务（端口：8080）
├── admin-frontend/       # Vue 3 管理后台（端口：3000）
├── mobile-frontend/      # H5 移动端（端口：5173）
├── miniprogram/          # 微信小程序（护工端）
├── family-miniprogram/   # 微信小程序（家属端）
└── docs/                 # 项目文档
```

### 技术栈详情

**后端：**
- 框架：Gin
- ORM：GORM
- 数据库：PostgreSQL 15
- 缓存：Redis 7
- 认证：JWT

**前端（管理后台）：**
- 框架：Vue 3.4 + TypeScript
- 构建：Vite 5.0
- UI：Element Plus 2.5
- 状态管理：Pinia
- 路由：Vue Router 4.2

**小程序：**
- 微信小程序原生框架
- TypeScript

## 🚨 需要解决的问题

### 1. Docker 未安装
- **影响：** 无法使用 docker-compose 快速启动
- **解决方案：** 使用 start.sh 脚本手动启动服务

### 2. 数据库未初始化
- **影响：** 无法正常使用系统
- **解决方案：** 需要执行数据库迁移脚本

### 3. 前端依赖未安装
- **影响：** 前端无法启动
- **解决方案：** 需要运行 npm install

### 4. Go 环境检查
- **影响：** 后端可能无法运行
- **解决方案：** 检查 Go 版本和依赖
